# Componentes UI

EmpiezaTuSaaS incluye componentes pre-construidos basados en shadcn/ui y Radix UI.

## Stack de UI

- **shadcn/ui**: Componentes accesibles y personalizables
- **Radix UI**: Primitivos sin estilos
- **Tailwind CSS**: Estilos utilitarios
- **Lucide Icons**: Iconos SVG

## Agregar más componentes

shadcn/ui tiene muchos más componentes disponibles:

```bash
npx shadcn@latest add [component]
```

Componentes populares:

```bash
npx shadcn@latest add alert
npx shadcn@latest add avatar
npx shadcn@latest add dropdown-menu
npx shadcn@latest add select
npx shadcn@latest add toast
npx shadcn@latest add table
```

Ver todos en [ui.shadcn.com/docs/components](https://ui.shadcn.com/docs/components).
