## Checklist pre-deploy

- [ ] Variables de entorno configuradas
- [ ] Base de datos en producción lista
- [ ] Webhooks de Stripe apuntando a producción
- [ ] Dominio verificado en Resend
- [ ] OAuth callbacks actualizados
- [ ] Tests pasando
- [ ] Build sin errores
