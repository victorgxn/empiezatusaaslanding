# Pagos con Stripe

Stripe es la plataforma de pagos más popular para SaaS. EmpiezaTuSaaS incluye integración completa.

## Funcionalidades incluidas

- **Checkout**: Páginas de pago hosted por Stripe
- **Suscripciones**: Planes mensuales/anuales con trial
- **Webhooks**: Sincronización automática de estados
- **Customer Portal**: Gestión de suscripciones por el usuario
- **Múltiples monedas**: EUR, USD, y más

## Modo de prueba

Stripe ofrece un modo de prueba completo. Usa estas tarjetas:

| Tarjeta | Resultado |
|---------|-----------|
| `4242 4242 4242 4242` | Pago exitoso |
| `4000 0000 0000 9995` | Pago rechazado |
| `4000 0025 0000 3155` | Requiere autenticación |
