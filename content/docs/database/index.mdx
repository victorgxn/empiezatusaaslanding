# Base de Datos

EmpiezaTuSaaS soporta dos opciones de base de datos para adaptarse a diferentes necesidades.

## Comparativa

| Característica | PostgreSQL + Prisma | Supabase |
|----------------|---------------------|----------|
| **Setup** | Manual | Automático |
| **Control** | Total | Limitado |
| **Migraciones** | Prisma Migrate | Dashboard/SQL |
| **ORM** | Prisma (type-safe) | Cliente JS |
| **Hosting** | Cualquiera | Supabase |
| **Costo** | Variable | Gratis hasta límites |
| **Realtime** | Requiere setup | Incluido |
| **Edge Functions** | No incluido | Incluido |

## Cuándo elegir cada opción

### Elige PostgreSQL + Prisma si:

- Quieres control total sobre tu base de datos
- Necesitas migraciones versionadas
- Prefieres un ORM con tipos completos
- Quieres hostear en cualquier proveedor
- Tu equipo tiene experiencia con SQL/ORMs

### Elige Supabase si:

- Quieres setup instantáneo
- Necesitas realtime out of the box
- Prefieres una UI para gestionar datos
- Quieres edge functions incluidas
- Estás construyendo un MVP rápido

<Callout type="info">
  Ambas opciones usan PostgreSQL como base de datos. La diferencia está en cómo interactúas con ella.
</Callout>
