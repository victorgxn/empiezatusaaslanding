# Supabase Database

Supabase proporciona una base de datos PostgreSQL completa con API REST automática, realtime y más.

## Características

- **PostgreSQL completo**: Todas las features de Postgres
- **API REST automática**: PostgREST genera API de tus tablas
- **Realtime**: Suscríbete a cambios en tiempo real
- **Dashboard visual**: UI para gestionar tablas
- **SQL Editor**: Ejecuta queries directamente
- **Backups automáticos**: En planes de pago
