# Social Login

Supabase facilita la configuración de OAuth providers desde el dashboard.

## Providers disponibles

Supabase soporta más de 20 providers OAuth:

- Google
- GitHub
- Apple
- Discord
- Twitter
- Facebook
- LinkedIn
- Microsoft
- Slack
- Spotify
- Y más...

## Configuración

<Tabs items={['Google', 'GitHub']}>
  <Tab value="Google">
    <Steps>
      <Step>
        ### Crear credenciales en Google

        1. Ve a [Google Cloud Console](https://console.cloud.google.com/)
        2. Crea o selecciona un proyecto
        3. Ve a **APIs & Services > Credentials**
        4. **Create Credentials > OAuth client ID**
        5. Selecciona **Web application**
      </Step>

      <Step>
        ### Configurar URIs

        **Authorized redirect URIs:**
        ```
        https://[TU-PROJECT-REF].supabase.co/auth/v1/callback
        ```

        <Callout type="info">
          Encuentra tu Project Ref en **Settings > General**
        </Callout>
      </Step>

      <Step>
        ### Configurar en Supabase

        1. Ve a **Authentication > Providers**
        2. Habilita **Google**
        3. Pega **Client ID** y **Client Secret**
        4. Click en **Save**
      </Step>
    </Steps>
  </Tab>

  <Tab value="GitHub">
    <Steps>
      <Step>
        ### Crear OAuth App en GitHub

        1. Ve a [GitHub Developer Settings](https://github.com/settings/developers)
        2. **New OAuth App**
        3. Llena los campos:
           - **Application name**: Tu App
           - **Homepage URL**: `https://tuapp.com`
           - **Authorization callback URL**: `https://[TU-PROJECT-REF].supabase.co/auth/v1/callback`
      </Step>

      <Step>
        ### Configurar en Supabase

        1. Ve a **Authentication > Providers**
        2. Habilita **GitHub**
        3. Pega **Client ID** y **Client Secret**
        4. Click en **Save**
      </Step>
    </Steps>
  </Tab>
</Tabs>

## Callback handler

```typescript title="app/auth/callback/route.ts"
import { createClient } from "@/lib/supabase/server";
import { NextResponse } from "next/server";

export async function GET(request: Request) {
  const { searchParams, origin } = new URL(request.url);
  const code = searchParams.get("code");
  const next = searchParams.get("next") ?? "/dashboard";

  if (code) {
    const supabase = await createClient();
    const { error } = await supabase.auth.exchangeCodeForSession(code);

    if (!error) {
      return NextResponse.redirect(`${origin}${next}`);
    }
  }

  // Error en autenticación
  return NextResponse.redirect(`${origin}/login?error=auth`);
}
```
