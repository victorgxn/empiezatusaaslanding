# Supabase Auth

Supabase Auth proporciona autenticación completa como servicio, integrada con tu base de datos Supabase.

## Por qué Supabase Auth

- **Setup instantáneo**: Funciona desde el minuto cero
- **UI incluida**: Dashboard para gestionar usuarios
- **Integración nativa**: Con database, storage y realtime
- **Row Level Security**: Políticas de acceso a nivel de fila
- **Múltiples providers**: OAuth, email, phone built-in

## Arquitectura

Supabase Auth funciona junto con:

- **PostgreSQL**: Tu base de datos
- **GoTrue**: El servidor de autenticación
- **PostgREST**: API REST automática
- **Realtime**: Websockets para datos en vivo
