# Autenticación

EmpiezaTuSaaS ofrece dos opciones de autenticación para adaptarse a diferentes necesidades y preferencias.

## Comparativa

| Característica | Better Auth | Supabase Auth |
|----------------|-------------|---------------|
| **Setup** | Medio | Rápido |
| **Control** | Total | Limitado |
| **Vendor lock-in** | Ninguno | Moderado |
| **Social login** | Config manual | Dashboard UI |
| **Magic links** | Plugin | Incluido |
| **2FA** | Plugin | Incluido |
| **Row Level Security** | Manual | Incluido |
| **Costo** | Solo DB | Gratis hasta límites |

## Cuándo elegir cada opción

### Elige Better Auth si:

- Quieres control total sobre la autenticación
- Necesitas personalización avanzada
- Prefieres evitar vendor lock-in
- Ya tienes una base de datos PostgreSQL
- Necesitas compliance específico (GDPR, etc.)

### Elige Supabase si:

- Quieres el setup más rápido posible
- Prefieres una UI para gestionar usuarios
- Necesitas storage y realtime además de auth
- Estás construyendo un MVP
- No quieres gestionar infraestructura

<Callout type="info">
  Puedes cambiar de una opción a otra más adelante, aunque requiere migración de datos.
</Callout>

## Funcionalidades incluidas

Ambas opciones incluyen:

- Email y password
- Social login (Google, GitHub, etc.)
- Magic links
- Verificación de email
- Recuperación de contraseña
- Sesiones seguras
- Protección CSRF

### Funcionalidades adicionales con plugins

Better Auth soporta plugins para:

- Two-Factor Authentication (2FA)
- Passkeys
- Username auth
- Phone/SMS auth
- Organizaciones y equipos

### Funcionalidades adicionales en Supabase

- Row Level Security (RLS)
- Hooks de autenticación
- Edge Functions
- Integración con Storage
