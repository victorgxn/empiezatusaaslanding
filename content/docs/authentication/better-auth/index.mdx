# Better Auth

Better Auth es una librería de autenticación framework-agnostic para TypeScript que ofrece control total sobre tu sistema de auth.

## Por qué Better Auth

- **Framework-agnostic**: Funciona con Next.js, Remix, Astro y más
- **Type-safe**: TypeScript first con inferencia completa
- **Extensible**: Sistema de plugins para funcionalidades adicionales
- **Sin vendor lock-in**: Tus datos en tu base de datos
- **Seguro**: Sigue mejores prácticas de seguridad

## Stack recomendado

| Componente | Tecnología |
|------------|------------|
| Auth | Better Auth |
| Database | PostgreSQL |
| ORM | Prisma |
| Hosting DB | Neon, Supabase DB, Railway |

<Callout type="info">
  Better Auth soporta múltiples bases de datos, pero recomendamos PostgreSQL por su robustez y compatibilidad.
</Callout>
