# Bienvenido a EmpiezaTuSaaS

El boilerplate definitivo para **indie hackers** y desarrolladores que quieren lanzar su SaaS sin perder tiempo en configuraciones repetitivas.

## Stack Tecnológico

- **Framework**: Next.js 16 con App Router
- **Estilos**: Tailwind CSS v4 + shadcn/ui
- **Autenticación**: Better Auth o Supabase Auth
- **Base de datos**: PostgreSQL con Prisma o Supabase
- **Pagos**: Stripe
- **Emails**: Resend + React Email
- **Deployment**: Vercel

## Empezar ahora

```bash
git clone https://github.com/tu-usuario/empiezatusaas.git
cd empiezatusaas
npm install
cp .env.example .env.local
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) y comienza a construir tu SaaS.
