# Emails con Resend

Resend es una API de email moderna diseñada para desarrolladores. Combinado con React Email, puedes crear templates hermosos con React.

## Por qué Resend

- **Developer-first**: API simple y moderna
- **React Email**: Crea templates con componentes React
- **Alta deliverability**: Emails que llegan a inbox
- **Dashboard**: Monitorea envíos y bounces
- **Precio justo**: 3,000 emails/mes gratis
